## 속성 목록  

<sup><sub>[*] 속성에 유형이 없는 것은 여러 유형 또는 다른 형식/패턴을 가질 수 있기 때문입니다</sub></sup>.  
- `alternateName[string]`: 이 항목의 대체 이름  - `connectionParameters[object]`: 시스템과의 연결을 실행하기 위한 매개변수  	- `port`:     
	- `server`:     
	- `user`:     
- `dataProvider[string]`: 조화된 데이터 엔티티의 공급자를 식별하는 일련의 문자  - `dateCreated[date-time]`: 엔티티 생성 타임스탬프. 이는 일반적으로 스토리지 플랫폼에서 할당합니다.  - `dateModified[date-time]`: 엔티티의 마지막 수정 타임스탬프입니다. 이는 일반적으로 스토리지 플랫폼에서 할당합니다.  - `description[string]`: 이 항목에 대한 설명  - `dueDate[date-time]`: 처리 실행에 필요한 예상 데이터/시간(이후 버전의 경우 계획된 처리를 허용하기 위해)  - `id[*]`: 엔티티의 고유 식별자  - `name[string]`: 이 항목의 이름  - `objective[string]`: 처리 목적  - `owner[array]`: 소유자의 고유 ID를 참조하는 JSON 인코딩된 문자 시퀀스가 포함된 목록입니다.  - `refMLModel[*]`: 처리에 사용할 머신 러닝 모델을 대상으로 하는 관계  - `refSubscriptionQuery[*]`: 머신 러닝 모델에서 사용할 구독 쿼리를 타깃팅하는 관계  - `seeAlso[*]`: 항목에 대한 추가 리소스를 가리키는 URL 목록  - `source[string]`: 엔티티 데이터의 원본 소스를 URL로 제공하는 문자 시퀀스입니다. 소스 공급자의 정규화된 도메인 이름 또는 소스 개체에 대한 URL을 사용하는 것이 좋습니다.  - `type[string]`: NGSI 엔티티 유형. MLProcessing이어야 합니다.  <!-- /30-PropertiesList -->
